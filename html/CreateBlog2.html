Jekyll없이 깃허브 블로그 만들기 Part 2

2024.12.07

1. 라이브서버 설치

실시간으로 편하게 확인하기 위해선 라이브서버를 설치할 필요가 있다

1-1. Node.js를 다운받고 설치한다

1-2. cmd를 실행하고 npm i -D live-server 를 입력해서 설치한다

1-3. 폴더위치에서 cmd로 입력하고 npx live-server를 입력하면 실행 된다

2. 비주얼 스튜디오 인코딩 설정

인코딩 설정이 잘못되면 문자가 깨질수있다 생성되는 파일들을 utf-8로 설정한다

2-1. 폴더에 .editorconfig 파일을 만든다

2-2.
root = true
[*]
charset = utf-8
입력하면 된다

3. 코드 텍스트로 보기

코드를 텍스트로 보고싶으면

3-1. <pre id="code-block"></pre> <script> const code = ` 이 안에 코드를 넣으면된다 `; document.getElementById('code-block').textContent = code; </script> 


        


        
Tip

        
깃허브와 라이브서버 둘다 적용시키기 편하게 상대경로를 사용하자

        
구현된걸 확인할땐 캐시를 지우기 위해서 Ctrl + Shift + R, Shift + F5를 사용하자

        
깃허브는 커밋후에도 사이트에 적용이 바로 안될수있다 라이브서버에서 성공했다면 기다리면 된다
