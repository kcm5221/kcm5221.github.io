RunTimer – 러닝 & 인터벌 타이머

개발 기간 2025.07.17 ~ 2025.08.25

▶ GitHub 저장소

프로젝트 개요

RunTimer는 알람과 인터벌 타이머, 러닝 모드(GPS) 기능을 하나로 합친 앱입니다.
아침 알람부터 운동/공부 루틴까지 관리할 수 있고, 달리기를 켜면 거리, 속도, 칼로리까지 자동 기록됩니다.
인터넷 없이도 완전히 동작하며, 모든 데이터는 내 기기 로컬에 저장됩니다.

주요 기능

  • 알람 – 요일/시간 설정, 중복 방지, 진동·소리 선택
  • 인터벌 타이머 – 워밍업·운동·휴식 단계를 직접 설정, 반복 실행
  • 러닝 모드 – GPS 기반 거리·평속·칼로리 계산, 경로 저장
  • 세션 기록 – 완료/중단된 모든 기록을 히스토리에서 확인
  • 설정 – 언어/테마, 단위(km·mi, kcal·kJ), 체중 입력, 센서(만보기·나침반) 사용 여부

UI 흐름

  • 홈 화면 – 오늘 울릴 알람 / 전체 알람을 한눈에
  • 인터벌 실행 화면 – 현재 단계, 남은 시간, 세트 진행률 표시
  • 러닝 모드 – 지도는 없지만 거리/속도 계산 + 기록 저장
  • 설정 화면 – 단위, 체중, 언어, 테마 변경 가능

개발 포인트

  • Flutter 3.x 단일 코드로 iOS/Android 동시 지원
  • 알람은 flutter_local_notifications + timezone으로 정확히 울림
  • 상태 관리는 Riverpod, 데이터는 Hive/SharedPreferences 로컬 저장
  • GPS는 geolocator 패키지로 5초 간격 기록
  • 로그는 LogService로 일관되게 남기고, Sentry로 릴리즈 오류 자동 보고

문제 해결 사례

  • 중복 알람 방지 – 같은 시각에 알람을 2개 이상 저장하지 못하도록 예외 처리
  • 배터리 절약 – GPS 업데이트 주기를 상황에 맞게 조정
  • UI 오버플로우 – ListTile trailing을 고정폭으로 교체
  • 안정성 – 비동기 처리 후 항상 context.mounted 체크

회고

러닝 앱은 실시간 처리 + 백그라운드 안정성이 핵심이었습니다.
알람·타이머·GPS를 각각 모듈화해 안정성을 확보했고, Flutter의 단일 코드베이스 덕분에 빠르게 두 플랫폼을 동시에 개발할 수 있었습니다.
아직 iOS 빌드는 테스트하지 못했지만, Android APK는 정상 실행됩니다.
