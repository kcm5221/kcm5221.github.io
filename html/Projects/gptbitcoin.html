gptbitcoin Trading Bot

개발 기간 2025.05 ~ 2025.06

깃허브 링크

1. 프로젝트 개요

gptbitcoin은 AI 바이브코딩 방식으로 구현한 파이썬 자동 매매 봇입니다.
전체 코드 작성은 AI의 도움을 받았으며, 저는 실행·테스트·구조 이해·실험을 담당했습니다.
Rule + GPT-4o AI 결정을 조합해 자동 매매를 수행하며, 모듈화를 통해 테스트·배포 자동화를 쉽게 설계했습니다.

2. 핵심 구현

2-1. AI 패턴 의사결정 & 반성문

# trading_bot/ai_helpers.py
async def ask_gpt(prompt: str) -> str:
    response = openai.ChatCompletion.acreate(
        model="gpt-4o-mini",  # 비용 절감용
        messages=[{ "role": "user", "content": prompt }],
        temperature=0.2,
    )
    return response.choices[0].message.content

최근 100봉 + 지표를 JSON으로 GPT-4o에 보내 buy / sell / hold 결정을 받고,
토큰 비용 절감을 위해 신호가 없을 땐 로컬 룰만 사용하도록 AI 로직을 구성했습니다.

2-2. 주문 실행 & 리스크 관리

# trading_bot/executor.py
qty = calc_position_size(balance_krw, atr, risk=BASE_RISK)

3. 문제 해결 & 개선

  • API Rate Limit 초과 → 캐시 TTL·백오프 로직 추가
  • AI 비용 폭증 → 신호 없을 시 룰 전략만 구동, 프롬프트 축소
  • 데이터 누락 및 오류 → REST 백업 엔드포인트 & SQLite 로그 검증

4. 배운 점

- AI가 생성한 코드를 실행하고 수정하면서 비동기 요청, API 연동, 리스크 관리 개념을 익혔습니다.
- 실제 거래소 API를 사용해 본 경험으로 실전 환경에서의 예외 처리와 비용 최적화의 중요성을 배웠습니다.
