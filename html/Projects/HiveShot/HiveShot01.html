HiveShot

기간: 2025.10.29 ~ 2025.11.15(예정)

링크

프로젝트 개요

게임명: HiveShot

장르: 자동 전투 디펜스

플랫폼: 토스

목표 유저층: 토스 사용자

플레이타임 목표: 1회 평균 4~6분

핵심 차별화 요소

  • 직접 조작 없이 전투가 진행되는 자동 공격 기반 전략형 디펜스
  • 매 판마다 다른 빌드와 강화 조합으로 전략적 선택의 재미

게임 시스템

  • 전투 구조: 하이브에서 적이 지속적으로 생성 → 플레이어 자동 공격
  • 웨이브 구성: 총 5웨이브, 각 웨이브 종료 시 난이도 상승
  • 경험치 및 강화: 적 처치 시 경험치 획득 → 레벨업 시 3가지 강화 중 선택
  • 체력 시스템: 플레이어 HP가 0이 되면 게임오버, 하이브 파괴 시 클리어
  • 재화: 적 처치 시 일정 확률로 드랍, 게임 종료 후 로비에서 누적
  • 티켓 시스템: 1판당 1장 소모 / 10분마다 1장 회복 / 광고 시 1장 추가
  • 상점 시스템: 로비에서 영구 강화 구매 가능

게임 루프 구조

  1. 로비: 티켓 보유 확인 → 게임 시작
  2. 플레이: 자동 전투 / 경험치 획득 / 강화 선택
  3. 웨이브 종료: 적 정리 후 다음 웨이브 진행
  4. 하이브 변화 → 최종 파괴 연출 (낮으로 전환)
  5. 결과 화면: 클리어 / 오버 판단 및 보상 정산
  6. 로비 복귀 → 재도전 또는 상점 접근
