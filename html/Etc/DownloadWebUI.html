2024.12.24

1. WebUI 설치

1-1. 엔비디아 그래픽카드 사용시

https://github.com/AUTOMATIC1111/stable-diffusion-webui/releases/tag/v1.0.0-pre 에서 sd.webui.zip을 다운로드
run.bat을 실행
설치 도중 File “~\launch.py”, line 325 나
ERROR: Could not find a version that satisfies the requirement pytorch_lightning==1.7.6 (from versions
이런 에러가 발생하면 update.bat을 실행, 업데이트 후 run.bat을 실행해주면 된다

1-2. 그 외 설치법

먼저 파이썬을 설치해야 하는데 3.10.6 버전을 설치 한다
(그 위의 버전의 경우 안됨)
파이썬 설치 중 Add Python 3.10 to PATH를 체크 후, 설치를 진행
다음은 git을 설치, 명령 프롬프트 실행 후 설치를 원하는 경로를 지정해주고
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git 를 입력한다
그러면 stable-diffusion-webui 폴더가 생성되고, webui-user.bat를 실행하면 된다
그 후 https://civitai.com/models 여기서 원하는 정보를 찾아보면서 그림을 생성하면 된다
