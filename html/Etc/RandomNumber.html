랜덤 숫자 생성하기

2025.01.16

입력한 숫자 범위(0부터 입력된 숫자까지)에서 랜덤 숫자를 생성

1. 작동 방식

  1. 사용자가 입력란에 숫자를 입력
  2. 버튼을 누르면 generateRandomNumber() 함수가 실행됨
  3. JavaScript가 입력값을 확인한 후, Math.random()과 Math.floor()를 사용하여 랜덤 숫자를 생성
  4. 결과가 페이지에 표시된다.

2. HTML 구조

HTML에 입력란과 버튼을 추가하고 결과를 표시할 영역을 만든다.

<label for="max-number">숫자입력:</label>
<input type="number" id="max-number" placeholder="Enter a number">
<button onclick="generateRandomNumber()">생성</button>
<div class="result" id="result"></div>

3. CSS


.result {
    font-size: 20px;
    color: blue;
    margin-top: 20px;
}

4. JavaScript

JavaScript는 입력된 숫자를 가져와 랜덤 숫자를 계산하고 결과를 화면에 표시한다.

function generateRandomNumber() {
    const maxNumber = parseInt(document.getElementById('max-number').value);

    if (isNaN(maxNumber) || maxNumber < 0) {
        document.getElementById('result').textContent = "올바른 0 이상의 숫자를 입력해주세요.";
        return;
    }

    const randomNumber = Math.floor(Math.random() * (maxNumber + 1));
    document.getElementById('result').textContent = `랜덤 숫자: ${randomNumber}`;
}
